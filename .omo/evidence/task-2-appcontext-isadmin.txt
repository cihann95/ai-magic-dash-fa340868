Task 2 - AppContext isAdmin State
==================================

AppContext.tsx updated with isAdmin state:

- State: const [isAdmin, setIsAdmin] = useState(false)
- useEffect: user değişince supabase.rpc("has_role", { _user_id: user.id, _role: "admin" }) çağrılıyor
